## 手順
1. 以下のコマンドで必要なパッケージをインストールします。 
```
npm install
```

2. 以下のコマンドでサーバーを起動します。
```
node index.js
```

3. localhost:3000へアクセスしてサーバーが立ち上がっていることを確認します。

4. 複数のブラウザを立ち上げで、チャット内容が同期されることが確認できます！
